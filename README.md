# Welcome Panel Tools

SillyTavern extension that applies welcome-screen UI tweaks without bundling core source files.

## What it does

- Removes legacy welcome-panel expand/collapse header buttons (if present)
- Forces tabs to stay visible (`recent chats` / `All Chats`)
- Normalizes `recent chats` tab label and style
- Slightly shifts all-chats group toggle arrow left for alignment

## Files to publish

Publish only this folder:

- `manifest.json`
- `index.js`
- `style.css`
- `README.md`
